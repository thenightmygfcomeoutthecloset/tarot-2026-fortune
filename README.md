# 2026下半年生辰八字运势测算 (BaZi Fortune 2026)

基于传统四柱八字（四柱命理）的 2026 丙午年下半年运势测算 Web 应用。

## 🌟 核心亮点

- **完整八字排盘引擎** (`bazi-engine.js`)
  - 年柱、月柱、日柱、时柱（四柱）计算
  - JDN 儒略日数精准日干支推导 & 五虎遁月 / 五鼠遁时
  - 天干地支藏干、五行生克与阴阳属性
  - 五行能量分布与日主强弱判断（身强/身弱、喜用神/忌神）
- **2026 下半年逐月运势分析** (`fortune-analyzer.js`)
  - 覆盖 2026 丙午年 7月 至 12月 六个月流月数据（乙未月 → 庚子月）
  - 五大核心维度评估：**事业运、财运、感情运、健康运、贵人运**
  - 丰富的十神解读文案库，针对不同日主生成个性化运势报告
  - 开运指南：幸运颜色、吉利方位、幸运数字、开运食物
- **现代星空视效与动画 UI**
  - Canvas 动态星空粒子与流星效果
  - 玻璃拟态卡片 (Glassmorphism) 与中国红/金色调融合
  - Canvas 纯原生绘制贝塞尔曲线运势走向图表
  - 响应式设计，完美适配移动端与 PC 端

## 🛠️ 项目结构

```text
.
├── index.html           # 页面结构
├── style.css            # 沉浸式星空主题样式
├── app.js               # 前端交互、Canvas 绘制与动画
├── bazi-engine.js       # 八字排盘核心逻辑引擎
└── fortune-analyzer.js  # 运势评分与文案生成模块
```

## 🚀 快速启动

无需任何打包与复杂环境设置，直接使用任意静态服务器打开：

```bash
# 使用 Python 启动本地服务
python -m http.server 8080

# 或使用 npx serve
npx serve .
```

在浏览器访问 `http://localhost:8080` 即可开始体验。
